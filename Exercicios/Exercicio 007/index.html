Principais Formatacoes

Negrito / destaque

E possivel temos um termo em negrito usando a tag B. Porem a tag B nao é semantica e nao e recondada para o uso, pois ela ja é antiga.

A forma certa e atualizada de se deixar um termo ou trecho em negrito é usando a tag "strong" que é uma tag semantica.

Italico / Enfase

Nesta frase temos um termo em italico usando a tag "i" que nao e semantica

A forma certa para deixarmos um termo ou trecho em italico é usando a tag EM

podemos tambem criar um texto marcado usando a tag MARK

Texto grande e texto pequeno

Estamos criando um texto grande usando a tag BIG e um texto pequeno. usando a tag SMALL.

Texto excluido

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado usando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois e para isso usamos a tag INS.

Textos sobreescritos

Para fazer um numero ou letra ficar acima como no exemplo: x20+3 e usada a tag SUP.

Para fazer um numero ou letra ficar abaixo como no exemplo: H2O é usada a tag SUP.
