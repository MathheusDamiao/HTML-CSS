Paragrafos e quebras de linhas

Para criar paragrafos eu uso a letra <p> e </p> para adicionar uma linha horizontal eu uso o comando <hr> e para adicionar um titulo eu uso o comando <h1> e </h1>

Se voce precisar quebrar o texto em algum lugar especifico
como este por exemplo use a tag <br>.
